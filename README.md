# ASP.NET MVC 5 -Video-Rental-Store

A movie store management system developed using ASP.NET MVC 5.

## Technologies
- ASP.NET MVC 5
- AutoMapper
- Autofac
- Entity Framework 6
- Microsoft SQL Server
- Bootbox
- Bootstrap 3
- jQuery
- jQueryDataTables
- Toastr
- Typeahead
